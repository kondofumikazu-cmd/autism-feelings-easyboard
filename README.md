# みえる医療指示ボード

病院・診療所で、医師・看護師が場面ごとの指示を選ぶと、患者さんに大きな文字とふりがなで見せられる静的ウェブページです。

## 特徴
- `index.html` 1ファイルだけで動作
- HTML / CSS / JavaScript を1つに同梱
- GitHub Pages にそのまま配置可能
- 通常時：橙色背景・黒文字
- ダークモード時：灰色背景・黄色文字
- ヒラギノ系フォント優先
- ふりがな付きの定型文を多数収録
- 日本語のみ / 日英併記を切り替え可能
- 文字サイズを大きく変更可能
- 自由入力文もその場で表示可能
- 個人情報の送信なし（静的ページ）

## 使い方
1. `index.html` を GitHub リポジトリの一番上に置く
2. GitHub の **Settings** → **Pages** を開く
3. Source を **Deploy from a branch** にする
4. Branch を **main**、フォルダを **/(root)** にする
5. 保存すると公開URLが発行される

## ローカルで試す方法
- パソコンで `index.html` をブラウザで開くだけでも動きます

## 注意
- このページは患者説明の補助ツールです
- 診断、同意取得、重要説明は、必要に応じて手話通訳、筆談、通訳機器、説明文書と併用してください
- 施設の運用に合わせて文言を調整してください

## 設計の参考にした研究の方向性
- 聴覚障害・難聴患者には、視覚的で確認可能なコミュニケーションが重要
- 短い文を段階的に示すことが有用
- 通訳や遠隔通訳だけでなく、文字・視覚情報の補助が必要
- 場面別の定型指示は、検査や処置時の不安軽減に役立つ
