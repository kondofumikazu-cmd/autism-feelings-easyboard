# いまの気持ちカード

自閉スペクトラム症の人が、いまの感情、困りごと、必要な支援を、
**大きな選択肢**と**自由入力**で伝えるための静的Webサイトです。

## ファイル

- `index.html`
- `styles.css`

この2つをそのまま GitHub にアップロードすれば動きます。
GitHub Pages でも公開できます。

## 特徴

- 日本語表示
- 白背景・黒文字の通常表示
- ダークモードでは黒背景・黄色文字
- ヒラギノ系フォント優先
- 大きめボタン
- 選択肢＋自由入力
- コピー機能
- テキスト保存機能
- 入力内容を `localStorage` に一時保存
- JavaScript は `index.html` の中に埋め込み済み

## 設計の考え方

このページは、次のような研究知見を踏まえて設計しています。

1. **AAC は、障害名そのものよりも、本人の困りごとや伝え方の特性に合わせて選ぶことが重要**とする文献レビュー。
2. **自閉症の人では、視覚的な状況提示があると感情語を選びやすい一方、状況から感情語を選ぶことに困難が見られる場合がある**という研究。
3. **シンボルや文字を使ったAACで、自発的コミュニケーションや相互的なやりとりが増えた**という事例報告。
4. **視覚的スケジュールやPECSのような視覚支援は、日常の見通しや要求表出の支えになりうる**という研究。
5. **本人の好みのコミュニケーション方法、感覚ニーズ、落ち着く方法、支援要望を記録して共有するツールは実用性がある**という自閉症フレンドリー医療のレビュー。
6. **OSのライト/ダーク設定をWebが尊重できる**という W3C の `prefers-color-scheme`。

## GitHub Pages で公開する方法

1. 新しいリポジトリを作る
2. この2ファイルをアップロードする
3. `Settings` → `Pages` → `Deploy from a branch`
4. `main` ブランチの `/root` を選ぶ
5. 保存すると公開URLが出ます
